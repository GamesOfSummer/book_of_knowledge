## Template literals ((Added in ES6))

`
let a = 5, b = 10;
`

console.log(\`Fifteen is ${a + b} and
not ${2 * a + b}.\`);


*is better than*

console.log('Fifteen is ' + (a + b) + ' and not ' + (2 * a + b) + '.');

**Another example** =>

* const age = 3
* console.log(\`I'm ${age} years old! \`)
